账户删除说明 / Account Deletion

最后更新 / Last updated: 2026-05-28

如何删除账户

  1. 打开 Corterm 应用
  2. 进入 设置 页面
  3. 滚动到底部，点击 "注销账户" 按钮
  4. 阅读确认提示，点击 "注销"
  5. 在第二次确认中再次点击 "注销"
  6. 账户删除完成后，您将自动返回登录页面

删除后的影响

注意：此操作不可逆。 账户删除后您将无法恢复。

账户删除后，以下数据将被处理：

  • 立即删除：所有终端会话记录
  • 立即解除：与 Worker 的绑定关系
  • 立即撤销：Apple ID 登录关联（如适用）
  • 标记为已删除：您的账户信息将被标记为 "deleted" 状态

删除后保留的数据

出于安全和审计目的，以下信息可能在有限时间内保留：

  • 审计日志（不含个人身份信息）

删除后的账户状态

账户删除后，您将无法使用该账户登录。关联的第三方登录（Apple、GitHub、Google）授权将被解除。如需重新使用服务，您需要创建新账户。

需要帮助？

如果您在删除账户过程中遇到问题，请联系 support@rwecho.top。
